# Agent Instructies

## Versiebeheer
Bij iedere aanpassing in de code moet het versienummer (`VERSION`) in `controller.py` worden aangepast volgens Semantic Versioning. De applicatie werkt `config.ini` automatisch bij bij het opstarten.

*   **Major (X.0.0):** Grote wijzigingen die niet backwards-compatible zijn.
*   **Minor (x.Y.0):** Nieuwe functionaliteit die backwards-compatible is.
*   **Patch (x.y.Z):** Bugfixes of kleine verbeteringen.
